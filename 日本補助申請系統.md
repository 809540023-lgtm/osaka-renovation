## 二、Agent #1：補助搜尋系統

### 功能說明
- 根據您的條件搜尋適用的補助制度
- 提供最新制度資訊
- 比較各制度優惠程度

### 搜尋關鍵字對話

```
用戶：我想搜尋大阪的裝修補助
Agent：好的！讓我幫您搜尋大阪適用的補助制度：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【大阪市】主要補助制度

1. 空家利活用改修補助
   └ 対象：空家・老朽住宅的耐震或節能改修
   └ 金額：最多¥1,000,000
   └ 申請窗口：大阪市

2. 耐震診断費用補助
   └ 対象：1981年以前建造的住宅
   └ 金額：最多¥50,000
   └ 申請窗口：大阪市

3. 省エネ改修支援
   └ 対象：節能設備更換（窗、空調、熱水器）
   └ 金額：最多¥300,000
   └ 申請窗口：大阪市

4. 地域まちづくり改修補助
   └ 対象：社區整體改善工程
   └ 金額：最多¥3,000,000
   └ 申請窗口：區公所

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【國民生活中央補助金】
- 住宅節能補助：最多¥200,000
- 長期優良住宅：最多¥500,000

請問您想了解哪個制度的詳細資訊？
```
